Un algoritmo genético (AG) es una técnica de optimización y búsqueda inspirada en la teoría de la evolución natural propuesta por Charles Darwin. 
Se basa en los principios de selección, cruza (recombinación) y mutación para encontrar soluciones aproximadas a problemas complejos donde los métodos tradicionales pueden resultar ineficientes.

Para configurar el código se necesita entrar a la clase main y modificar los parámetros, se tienen que definir las ciudades y configurar los parámetros del algoritmo genético, específicamente en la función ejecutar y posteriormente ejecutar el archivo main.py

Dependencias necesarias para correr el código:
.- Random
.- numpy
.- pandas
.- operator
